# CreatorHub

Plataforma SaaS para gestión y monetización de creadores de contenido.

## Stack

- Backend: NestJS
- Frontend: Angular
- Database: PostgreSQL
- Infra: Docker
- CI: GitHub Actions
- Code Quality: SonarCloud

## Requisitos

- Node (ver .nvmrc)
- Docker
- WSL2 (Windows)
